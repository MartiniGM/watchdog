To set up mysql for testing:

sudo apt-get install mysql-server
sudo apt-get install python-mysqldb

mysql -u root -p
enter password: raspberry

mysql> CREATE DATABASE demosdb;
Query OK, 1 row affected (0.02 sec)

mysql> CREATE USER 'demo-user'@'localhost' IDENTIFIED BY 'plaintext';
Query OK, 0 rows affected (0.00 sec) (NOTE: pass for demo-user is "plaintext"!)

mysql> USE demosdb;
Database changed

mysql> GRANT ALL ON demosdb.* TO 'demo-user'@'localhost';
Query OK, 0 rows affected (0.00 sec)

mysql> quit;
Bye


Then run mysqltest_create.py to create a sample test db...
mysqltest_set.py to populate it...
mysqltest_get.py to read the data back...
